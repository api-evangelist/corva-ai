# Corva AI (corva-ai)

Corva is a Houston-based AI software company that provides a real-time drilling, completions, geoscience, and sustainability analytics platform for the oil and gas industry. The Corva platform exposes a RESTful Data API and the Corva Dev Center SDK / app hosting environment that lets customers build frontend, backend, scheduled, and stream apps on top of Corva data assets.

## APIs

### Corva Data API

RESTful Data API for accessing Corva drilling and completions data, real-time streams, and analytics, documented via Swagger UI.

- [Documentation](https://api.corva.ai/documentation/index.html)
- [Dev Center Docs](https://dc-docs.corva.ai/)

### Corva Dev Center

SDK and app hosting environment with Python and JavaScript SDKs, UI components, app templates, and a public marketplace for custom applications on top of Corva data.

- [Documentation](https://dc-docs.corva.ai/)
- [Python SDK](https://dc-docs.corva.ai/docs/Backend/Software%20Development%20Kits/Python/)
